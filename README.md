## このページは何？

[noco株式会社](https://nocoinc.co.jp/) のデザイナー・ソフトウェアエンジニア採用（業務委託含む）に興味がある方向けの情報共有サイトです。


## noco株式会社について

当社は千葉県に本社を構えるIT・スタートアップです。インタネットテクノロジーを通じて、働く人と組織をエンパワーメントするデジタルワークプレイスツールの開発、提供を行っています。

今回、新クラウドサービスのロンチを予定しており、当社が次なるフェイズに進むにあたり必要なプロダクト開発を進めています。

nocoが大切にしているビジョン・ミッション、求めるバリューについて制定しています。

### ブランド・ステートメント
『チームで働くことを。なめらかに。』

### ミッション
『すべての働く人たちが、いきいきと活躍できる社会を創る』

### ビジョン
『組織で働く人たちが直面するリソースの課題を解決するため デジタルワークプレイスツールを提供する』

### Will
『プロダクト創り。事業創り。仲間創り。未来創り。』

### Credo

- 『ただしく、つよく。変化し続けることを楽しむ』
- 『誰かの嬉しいからはじめる。』
- 『逆算思考で課題を解決する』
- 『成功体験を進んでアンラーニングする』

## 業務内容

### 現在の技術スタック

- フロントエンド：TypeScript、React、Next.js、Storybook、OpenAPI、SWR、MUI
- サーバーサイド：Ruby on Rails

- ミドルウェア：Algolia

- モニタリング：Sentry
- DB：MySQL
- CI/CD: GitHub Actions
- プロジェクト管理：GitHub、ZenHub
- UI デザイン：Figma
- コミュニケーション：Slack、Notion
- その他業務ツール：Google Workspace、1Password

## ハードスキル（フロントエンド）

**MUST**

- TypeScript、React、Next.js、Storybook、OpenAPI を使った開発経験
- 実際にサービス開始された SPA のゼロからの設計経験
- Figma で作成されたデザインのマークアップ（HTML，CSS）

**SHOULD**

※ ジョイン後1ヶ月程度での習得を期待

- SWR もしくは Recoil を用いた開発経験
- MUI 、Ant Design もしくは Chakra UI での開発経験
- Mock Service Worker もしくは Prisma を用いた モックAPI 開発

**BETTER**

- デザインシステムの設計・開発経験

## 求める人物像

- コミュニケーションはチーム開発で最重要であると同時にコストであると認識し、効率的なプロダクト開発のためにはそれを改善し続けないといけないと考えるひと
- 新しい技術をキャッチアップするだけでなく、実際に業務に活かしたいと思うひと
- 細部のチューニングをしていると気づいたら没頭しているひと
- 既存の文献・ホワイトペーパー・論文などをキャッチアップし、守破離を意識して設計できるひと
- 経験主義に立脚した高速な試行錯誤に興味あるひと
